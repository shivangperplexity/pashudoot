API Documentation

Pashu Mitra uses two in-house developed deep learning models hosted on Hugging Face to perform its detection and classification tasks.

1. CattleLense (Detection)

This model is a simple image classifier that determines if an image contains a cattle.

  • **Endpoint:** https://rjsmoke00-cd.hf.space/predict
  • **Method:** POST
  • **Input:** FormData with an image file.
  • **Output:** A JSON object with a prediction key, which will be either "LABEL_0" (No Cattle) or "LABEL_1" (Cattle Detected).

2. Creto.ai (Classification)

This model classifies the breed of the cattle if it has been detected by the first model.

  • **Endpoint:** https://Shivangguptasih-v2.hf.space/predict
  • **Method:** POST
  • **Input:** FormData with an image file.
  • **Output:** A JSON object with a prediction key, which contains the name of the identified breed in lowercase.
